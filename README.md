# CodeAlpha - Network Sniffer

## 🔍 Project Title
**Basic Network Sniffer using Python & Scapy**  
A simple tool to capture and analyze live network packets.

## 🧠 What It Does
- Captures live packets on your system
- Displays:
  - Source IP
  - Destination IP
  - Protocol number
  - Timestamp

## 💻 Technologies Used
- Python 3.12
- Scapy
- IDLE
